```
var str = "qwertyuiop";
str.replace(/(we|ty|io)/gim, "!$&!");
// "q!we!r!ty!u!io!p"
```

```
import re
pattern = re.compile(r'(\d) (\d) (\d)')
match = pattern.search('1 2 3', pattern)
match.group(1, 2, 3)
# ('1', '2', '3')
```

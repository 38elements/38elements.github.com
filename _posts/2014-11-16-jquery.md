* jQueryで要素の作成
{% highlight javascript %}
var $foo = $("<div/>", {
    class: "bar",
    on: {
        click: function() { alert("foo") }
    },
    text: "bar"
});
{% endhighlight %}

* jQueryで要素に登録されているイベントとそのcallback関数の一覧を取得する
{% highlight javascript %}
$._data(<HTML要素>, "events");
{% endhighlight %}
